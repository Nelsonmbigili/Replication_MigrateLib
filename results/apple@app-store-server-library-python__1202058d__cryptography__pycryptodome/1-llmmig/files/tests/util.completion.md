### Explanation of Changes

To migrate the code from the `cryptography` library to the `pycryptodome` library, the following changes were made:

1. **Key Generation**:
   - Replaced `cryptography.hazmat.primitives.asymmetric.ec` with `Crypto.PublicKey.ECC` for generating elliptic curve keys.
   - Used `Crypto.PublicKey.ECC.generate()` to generate private keys and `export_key()` to export them in PEM format.

2. **Key Serialization**:
   - Replaced `cryptography.hazmat.primitives.serialization` with `Crypto.PublicKey.ECC` for exporting keys.
   - Used `export_key(format='PEM')` for private and public key serialization.

3. **JWT Signing and Verification**:
   - The `jwt` library remains unchanged for signing and verifying JWTs, as it is compatible with keys generated by `pycryptodome`.

4. **Removed Cryptography Imports**:
   - Removed all imports from `cryptography.hazmat.primitives`.

```python
from typing import Any, Dict
import jwt
from jwt.api_jwt import decode_complete
import json
import os

from Crypto.PublicKey import ECC
from appstoreserverlibrary.models.Environment import Environment

from appstoreserverlibrary.signed_data_verifier import SignedDataVerifier

def create_signed_data_from_json(path: str) -> str:
    data = read_data_from_file(path)
    decoded_data = json.loads(data)
    private_key = ECC.generate(curve='P-256').export_key(format='PEM')
    return jwt.encode(payload=decoded_data, key=private_key, algorithm='ES256')

def decode_json_from_signed_date(data: str) -> Dict[str, Any]:
    private_key = ECC.generate(curve='P-256')
    public_key = private_key.public_key().export_key(format='PEM')
    return decode_complete(jwt=data, key=public_key, algorithms=['ES256'], options={"verify_signature": False})

def read_data_from_file(path: str) -> str:
    full_path = os.path.join(path)
    with open(full_path, mode='r') as test_file:
        return test_file.read()
    
def read_data_from_binary_file(path: str) -> str:
    full_path = os.path.join(path)
    with open(full_path, mode='rb') as test_file:
        return test_file.read()

def get_signed_data_verifier(env: Environment, bundle_id: str, app_apple_id: int = 1234) -> SignedDataVerifier:
    verifier = SignedDataVerifier([read_data_from_binary_file('tests/resources/certs/testCA.der')], False, env, bundle_id, app_apple_id)
    verifier._chain_verifier.enable_strict_checks = False # We don't have authority identifiers on test certs
    return verifier

def get_default_signed_data_verifier():
    return get_signed_data_verifier(Environment.LOCAL_TESTING, "com.example")
```

### Summary of Changes
- Replaced `cryptography`'s elliptic curve key generation and serialization with `pycryptodome`'s `ECC` module.
- Removed all imports from `cryptography`.
- The rest of the code remains unchanged, ensuring compatibility with the larger application.
